lipandas
================
Load in Ipython Pandas: load a CSV file into a Pandas DataFrame and open it in IPython.


## Installation

```bash
$ pip install lipandas
```

## Usage
```bash
$ lipandas data.csv

DataFrame loaded. Use 'df' to access the data.
```
